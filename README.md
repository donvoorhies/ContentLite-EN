# ContentLite (EN)

Lightweight PHP-based content management with news, gallery, admin panel, and a first-run installer.

## Features

- News section with TinyMCE-based editing
- Gallery with albums and images
- First-run installer for database setup
- Table prefix support for shared databases

## Setup

1. Upload the files to your server.
2. Open `install.php` in the browser.
3. Fill in the database settings and, if needed, a table prefix.
4. Remove or protect `install.php` after installation.
